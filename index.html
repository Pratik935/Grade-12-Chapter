Chapter 1: Database Management System

Data

Data refers to raw facts, figures, or statistics that can be processed to produce meaningful information. Data can be in the form of numbers, text, images, audio, or video.

Information

Information is processed, organized, and structured data that provides context and meaning. It is useful for decision-making and understanding.

Features of Information

  • Accuracy: Information should be free from errors
  • Timeliness: Information should be available when needed
  • Relevance: Information should be pertinent to the user's needs
  • Completeness: Information should include all necessary details
  • Consistency: Information should be uniform across the system
